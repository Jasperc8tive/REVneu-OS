# Revneu OS
### Agentic Revenue Growth Platform for Nigerian Businesses

> Connect to a company's data → analyze growth performance → deploy AI agents that recommend and execute revenue optimizations.

| Stage | Focus | Status |
|-------|-------|--------|
| Stage 1 | Architecture & Foundation | ⬜ Not Started |
| Stage 2 | Core System (Auth, Multi-tenancy) | ⬜ Not Started |
| Stage 3 | Data Layer (Integrations) | ⬜ Not Started |

## Stack

- **Frontend:** Next.js 14 + TailwindCSS
- **Backend:** NestJS (Node.js)
- **AI Layer:** FastAPI (Python)
- **Database:** PostgreSQL (Prisma ORM)
- **Queue:** Redis + BullMQ
- **Payments:** Paystack + Stripe
- **Hosting:** Docker → AWS ECS
